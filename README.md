# UI-RPG
Šī ir vienkārša turn‑based cīņas spēle, kur spēlētājs ar uzbrukuma pogu cīnās pret nejauši ģenerētiem pretiniekiem. Katram pretiniekam tiek piešķirts nejaušs vārds, HP un attēls. Pēc spēlētāja uzbrukuma pretinieks automātiski uzbrūk atpakaļ. Spēlētājs var izvēlēties starp trim dažādiem ieročiem (Blade, Poison Blade, Bow), un katrs ierocis dod atšķirīgu bojājumu - tas ir mans izpildītais papildus uzdevums. Izvēlētais ierocis uzreiz parādās ekrānā. Kad pretinieks mirst, automātiski parādās jauns pretinieks. Kad spēlētāja HP sasniedz 0, tiek parādīts Game Over panelis ar Quit pogu, kas ļauj iziet no spēles. Spēlē ir UI elementi spēlētāja un pretinieka HP, vārdiem, attēliem, ieroču izvēlei un Game Over ekrānam. Projektā tika izmantoti OOP principi: mantošana un polimorfisms ieroču sistēmā (Weapon bāzes klase un trīs atvasinātās klases ar atšķirīgu uzvedību), kā arī enkapsulācija spēlētāja un pretinieka datu pārvaldībā.
